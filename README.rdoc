keyfree
=======
NAME OF APP: KEY FREE

SHORT DESCRIPTION: This app will list LA restaurants with a free or small corkage fee.

FEATURES:
*Search by zip code in the LA area
*Search by food type
*Search within corkage fee ranges (e.g.- free, <$5, $5-10, $10-15, >$15) 
*Find wine stores in your zip code
*Users can login and save or update corkage fee info
